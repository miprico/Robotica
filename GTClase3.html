TEORÍA

Circuitos, Multímetro y Protoboard

Lee los conceptos y fíjate en las diferencias clave entre los dos tipos de circuito

⏱ 15 min aprox.
🔗
Circuito en Serie
Componentes conectados uno tras otro en un solo camino. La corriente es la misma en todos, pero el voltaje se divide. Si uno falla, todo se apaga.
⑂
Circuito en Paralelo
Componentes en ramas independientes. El voltaje es el mismo en cada rama. Si uno falla, los demás siguen funcionando. Así funciona el hogar.
🔬
Multímetro
Mide voltaje (V), corriente (A), resistencia (Ω) y continuidad. Es la herramienta de diagnóstico fundamental en electrónica.
🧩
Protoboard
Placa de pruebas con orificios interconectados. Las filas horizontales del centro están conectadas internamente (5 orificios por fila). Los buses laterales son + y −.
⚡
Ley de Ohm
V = I × R — Voltaje = Corriente × Resistencia. Con Arduino 5V y 220Ω, la corriente en el LED es ≈22mA. Suficiente para encenderlo sin dañarlo.
💡
Secuenciador de LEDs
Varios LEDs conectados en protoboard que se encienden/apagan en secuencia programada con Arduino. Combina circuitos y código.

CÓDIGO Y PROYECTO

Secuenciador de LEDs 💡

Escribe el código, prueba el simulador y personaliza tu animación

⏱ 25–35 min aprox.
secuenciador_leds.ino
// ── Módulo 1 · Clase 3 · Secuenciador de LEDs ──────────────── int led1 = 11; // LED 1 (rojo) int led2 = 12; // LED 2 (amarillo) int led3 = 13; // LED 3 (verde) int espera = 300; // ms entre cada LED — ¡personaliza esto! void setup() { pinMode(led1, OUTPUT); pinMode(led2, OUTPUT); pinMode(led3, OUTPUT); } void loop() { // ── IDA: LED 1 → 2 → 3 ────────────────────── digitalWrite(led1, HIGH); delay(espera); digitalWrite(led1, LOW); digitalWrite(led2, HIGH); delay(espera); digitalWrite(led2, LOW); digitalWrite(led3, HIGH); delay(espera); digitalWrite(led3, LOW); // ── VUELTA: LED 3 → 2 → 1 ─────────────────── digitalWrite(led3, HIGH); delay(espera); digitalWrite(led3, LOW); digitalWrite(led2, HIGH); delay(espera); digitalWrite(led2, LOW); digitalWrite(led1, HIGH); delay(espera); digitalWrite(led1, LOW); }

QUIZ

¿Cuánto aprendiste?

7 preguntas sobre los temas exactos de la clase. ¡Sin trampa!

1. En un circuito en serie con 3 LEDs y fuente de 5V, ¿cuánto voltaje recibe cada LED aproximadamente?
2. En la protoboard, ¿cuántos orificios están conectados internamente en la zona central?
3. Estás probando un cable con el multímetro en modo continuidad. No suena el beep. ¿Qué significa?
4. En el circuito en paralelo, desconectas un LED. Los otros dos LEDs...
5. En el código del Secuenciador, ¿qué variable controla la velocidad de la animación?
6. Para medir voltaje con el multímetro, ¿cómo conectas las puntas al circuito?
7. ¿Cuántas resistencias de 220Ω necesitas para conectar 3 LEDs en paralelo al Arduino?
