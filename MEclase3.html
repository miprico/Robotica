teoría

Tipos de circuitos

🔗
Circuito en Serie
Los componentes se conectan uno tras otro formando un solo camino para la corriente. Si un componente falla, todo el circuito se interrumpe. El voltaje se divide entre los componentes, pero la corriente es la misma en todo el circuito.
⑂
Circuito en Paralelo
Los componentes se conectan entre los mismos dos puntos formando ramas independientes. Si un componente falla, los demás siguen funcionando. El voltaje es el mismo en cada rama, pero la corriente se divide.
🔬
Multímetro
Instrumento que mide voltaje (V), corriente (A) y resistencia (Ω). También sirve para medir la continuidad de un circuito (si hay paso de corriente). Es la herramienta de diagnóstico fundamental en electrónica.
🧩
Protoboard
Placa de pruebas con filas y columnas de orificios interconectados que permiten conectar componentes sin soldar. Las filas horizontales en el centro están conectadas internamente. Las columnas laterales (rojas y negras) son los buses de alimentación.
⚡
Ley de Ohm
Relaciona las tres magnitudes eléctricas fundamentales: V = I × R (Voltaje = Corriente × Resistencia). En un circuito con Arduino de 5V y una resistencia de 220Ω, la corriente que pasa por el LED es aproximadamente 22mA — suficiente para encenderlo sin dañarlo.
🔄
Secuenciador de LEDs
Circuito donde varios LEDs se encienden y apagan en secuencia programada. Combina el conocimiento de circuitos (conexiones de LEDs con resistencias en protoboard) con la programación Arduino (digitalWrite y delay) para crear animaciones de luz.

experimentos

Los 3 experimentos de la clase

1
Uso del multímetro

Aprender a usar el multímetro para medir voltaje, resistencia y continuidad en circuitos reales. Verificar el valor de resistencias con el código de colores.

  • Medir el voltaje de la protoboard conectada al Arduino (esperar 5V)
  • Medir la resistencia de una resistencia de 220Ω y verificar la lectura
  • Probar continuidad de un cable (debe sonar el beep)
  • Probar continuidad de un cable roto o doblado
  • Medir la caída de voltaje en un LED encendido (~2V)
2
Circuitos en serie y paralelo

Construir y comparar ambos tipos de circuito con LEDs. Observar la diferencia de brillo y verificar qué pasa cuando se desconecta un LED.

  • Conectar 3 LEDs en serie con 1 resistencia entre pin 13 y GND
  • Observar el brillo (más opaco por la división de voltaje)
  • Desconectar un LED — ¿qué pasa? (se apagan todos)
  • Conectar los mismos 3 LEDs en paralelo (cada uno con su resistencia)
  • Comparar el brillo con el circuito en serie — notar la diferencia
  • Desconectar un LED en paralelo — ¿los demás siguen? (sí)
3
Uso de protoboard

Construir el circuito del Secuenciador de LEDs directamente en la protoboard, conectando los 3 LEDs a los pines del Arduino con la lógica de conexión interna de la placa.

  • Conectar el cable de 5V al bus positivo (+) de la protoboard
  • Conectar GND al bus negativo (−) de la protoboard
  • Insertar 3 LEDs en la zona central — uno por fila
  • Agregar una resistencia de 220Ω en serie con cada LED
  • Conectar cada resistencia a un pin digital del Arduino (11, 12, 13)
  • Conectar el cátodo de cada LED al bus negativo (GND)
  • Subir el código del secuenciador y ver la animación

Código: Secuenciador de LEDs

secuenciador_leds.ino
// ─── Módulo 1 · Clase 3 · Secuenciador de LEDs ────────────── // LEDs conectados en protoboard — pines 11, 12, 13 int led1 = 11; // LED rojo int led2 = 12; // LED amarillo int led3 = 13; // LED verde int espera = 300; // ms entre cada LED void setup() { pinMode(led1, OUTPUT); pinMode(led2, OUTPUT); pinMode(led3, OUTPUT); } void loop() { // ── IDA: LED 1 → 2 → 3 ────────────────────── digitalWrite(led1, HIGH); delay(espera); digitalWrite(led1, LOW); digitalWrite(led2, HIGH); delay(espera); digitalWrite(led2, LOW); digitalWrite(led3, HIGH); delay(espera); digitalWrite(led3, LOW); // ── VUELTA: LED 3 → 2 → 1 ─────────────────── digitalWrite(led3, HIGH); delay(espera); digitalWrite(led3, LOW); digitalWrite(led2, HIGH); delay(espera); digitalWrite(led2, LOW); digitalWrite(led1, HIGH); delay(espera); digitalWrite(led1, LOW); }

claves

Tips para la clase

🧩
Protoboard: verifica filas
Cada fila de 5 orificios está conectada internamente. Nunca pongas dos pines de la misma fila que no deben tocarse.
🔬
Multímetro: rango correcto
Selecciona siempre el rango más alto y ve bajando. Nunca midas voltaje con la perilla en Ω (puede dañar el equipo).
💡
LEDs más brillantes
En paralelo cada LED recibe los 5V completos → brillo máximo. En serie el voltaje se divide → menos brillo.
⚡
Resistencia siempre
Aunque en paralelo cada LED funciona independientemente, CADA uno necesita su propia resistencia de 220Ω.
🔍
Diagnóstico rápido
¿LED no enciende? Usa el multímetro en modo diodo para verificar si el LED funciona antes de revisar el código.
🎯
Personaliza el proyecto
Cambia la variable "espera" para velocidades distintas. Prueba encender dos LEDs al mismo tiempo en algunos pasos.
